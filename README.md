# Discord - Fivem Moderation Bot

# Kurulum
* İlk olarak bilgisayarına [Node JS](https://nodejs.org/en/) indir.
* Bu projeyi zip halinde indir.
* Herhangi bir klasöre zipi çıkart.
* Daha sonra klasörün içindeki `config.json` dosyasının içindeki bilgileri doldur.
* Sonra klasörün içerisinde bir `powershell` ya da `cmd` penceresi aç.
* ```npm install``` yazarak tüm modülleri kur.
* Kurulum bittikten sonra `start.bat` adlı dosyaya çift tıkla ve botu başlat.

Tada 🎉. Artık moderasyon botun hazır. Dilediğin gibi kullanabilirsin.
